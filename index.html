DESENVOLVEDOR FULLSTACK - CLARO

Junho de 2024 - Atualmente

Do Ustore, um hub de TI da Claro, UCloud é uma plataforma integradora de serviços de nuvem, que permite administrar múltiplas nuvens como AWS, Azure, Google, entre outras. UCloud está em migração de monolítico para microsserviços. Trabalho para manter o bom funcionamento dos sistemas do UCloud em Java 8, 17 e 21 e de sua transição. Utilizamos também de metodologias ágeis, como Scrum e Kanban para tornar o processo mais eficiente.
Principais tecnologias utilizadas: AngularJS, Java, Spring Boot, Maven, Docker, RabbitMQ, Git, Gitlab, MySQL, jUnit e Mockito.

DESENVOLVEDOR BACKEND JAVA — XMOB

Janeiro de 2024 - Junho de 2024

Lirou Store é uma loja de óculos de sol com clientes nas proximidades que visa expandir o negócio nacionalmente. A construção e manutenção de um ecommerce no ambiente web para o estabelecimento foi minha parte na busca por esse objetivo. Entre os principais recursos do sistema estão:

DESENVOLVEDOR FULLSTACK — XMOB.

Junho de 2023 - Dezembro de 2023

Saboroso Burger era uma hamburgueria que geria seus atendimentos através das redes sociais. Visando uma melhor padronização na criação de pedidos para evitar erros e economizar tempo de atendimento, tive o desafio de criar uma aplicação que permitisse aos clientes:

  • Ver as opções disponíveis no cardápio
  • Fazerem seus pedidos com autoatendimento
  • Acompanhar as atualizações referentes a cada pedido

Aos administradores:

  • Gerir os itens disponíveis no cardápio
  • Gerir os pedidos
  • Notificar atualizações em cada pedido

Principais tecnologias utilizadas: Java com Spring Boot e Maven, Angular, Docker e Postgres, Git, Github e WebSocket.
